Anime Cavern is a website developed entirely in the newly released client side framework for .NET, ASP.NET Core Blazor Webassembly and pure CSS. Making use of the public anime APIs Jikan and AniList, the app uses the new IHttpClientFactory to create the HttpClient instances needed to make the calls to the APIs. The requests are abstracted behind two interfaces, one for each API. To deserialize the JSON content received, the new System.Text.Json takes care of the errand brilliantly.

E-commerce App
×

This is a scalable demo E-commerce web app built using ASP.NET Core for the back-end on .NET 5 and Angular and Bootstrap for the front-end. Redis was also used for caching and increased performance.
The back-end side followed the clean architecture pattern by separating the high level modules from the low level modules and having them both rely on abstractions instead of each other.
The Core project represent the abstractions and live in their own assembly without the need to depend on other projects. On the other hand, the Infrastructure project lives in its own assembly but depends on the Core project to access its interfaces and data models. The API project avoids relying on Infrastructure directly and solely depends on the Core project.
Stripe was also implemented for Payment processing and is working well. To test the payment process, please refer to Stripe's tests cards and DO NOT USE A REAL PAYMENT CARD. The project was containerized into separate Docker images and interconnected through Docker compose.
